à rendre pour le 18 Avril
Formation RSC Contrat Pro

--------------------------------------------
Fonctionnalités :

-Ajout billet
    Titre
    Contenu
    Date
    Tag
    publié ou non publié
-Suppression billet
-Edition billet

--------------------------------------------
On peut créer un billet, lui choisir une date et si la date est postérieur au jour J alors le billet ne sera pas afficher

On peut assigner un tag au billet, une fois le tag cré si on remets le meme tag pour un autre billet sera lié au tag par une table transmediaire.
Cela permet d'avoir qu'une vois le libellé present dans la table des tags.

La liste des tag est affichée à droite
